# 💰 SmartFin – Full Stack FinTech Application

## 🚀 Overview

SmartFin is a full-stack fintech application built using **Spring Boot, MySQL, HTML, CSS, and JavaScript**.

This project is designed around a real-world design thinking problem:

> **“How can borrowing money feel less intimidating and more transparent to customers?”**

The system focuses on simplifying financial decisions by providing **clear loan insights, EMI breakdowns, and risk indicators**.

---

## 🧠 Key Idea

Many users hesitate to take loans due to:

* Lack of transparency
* Confusing financial terms
* Hidden costs

SmartFin solves this by:

* Clearly showing **monthly EMI**
* Displaying **total payable amount**
* Providing **risk-based feedback**

## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/manoharan12105-beep/smartfin-fullstack-fintech.git
   ```

2. Open in your IDE (IntelliJ / VS Code)

3. Configure MySQL in `application.properties`:

   ```properties
   spring.datasource.url=jdbc:mysql://localhost:3306/smartfin
   spring.datasource.username=your_username
   spring.datasource.password=your_password
   ```

4. Run the Spring Boot application

5. Open frontend:

   * `login.html`
   * `register.html`
   * `dashboard.html`
